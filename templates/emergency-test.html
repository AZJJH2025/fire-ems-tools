What Is Emergency Mode?

The FireEMS resilience framework enables our tools to continue functioning even when infrastructure services are degraded or unavailable. Emergency Mode activates when the application detects problems with:

  • Static file loading (CSS, JavaScript)
  • API endpoints (data retrieval and storage)
  • Server connectivity (offline scenarios)

New: Chart.js Canvas Reuse Fix

We've implemented a robust solution to prevent "Canvas is already in use" errors when creating charts in emergency mode.

The ChartManager service properly handles the chart lifecycle through canvas regeneration, cleanup, and error recovery.
